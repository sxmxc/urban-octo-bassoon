# Backlog

A place to capture ideas that are interesting but not yet committed to the active roadmap. Once an idea is promoted into `TASKS.md`, remove it here so the planning docs stay unambiguous.

## Ideas
- Add realtime collaboration (shared endpoint editing) via WebSockets.
- Add request recording and replay for deployed routes and execution traces.
- Evolve the schema-studio canvas beyond schema nodes into route, data-source, and transform primitives once the `Vue Flow` foundation is proven.
- Add a connector marketplace once the first `HTTP` and `Postgres` connectors are operationally mature.
